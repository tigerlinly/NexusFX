# คู่มือผู้ใช้งาน (User Guide) - NexusFX Platform

ยินดีต้อนรับสู่ **NexusFX Trading Platform**! แพลตฟอร์มเทรดอัลกอริทึมอัตโนมัติและเครื่องมือการซื้อขาย (Multi-Terminal) ระดับสถาบัน
คู่มือฉบับนี้จะพาคุณเริ่มต้น **ทีละขั้นตอน (Step-by-Step)** คราวเดียวตั้งแต่การเข้าสู่ระบบไปจนถึงการปล่อยและควบคุมบอทตัวแรก!

---

## สรุป 6 ขั้นตอนสู่การเทรดอัตโนมัติ 🚀
1. **ล็อกอินเข้าใช้งานครั้งแรก (Login)**
2. **ผูกพอร์ตและเติมเงิน (Add Account / Wallet Deposit)**
3. **สมัครแพ็กเกจ (Subscriptions / Billing)** 
4. **ตามหากลยุทธ์ในความฝัน (Strategy Store)**
5. **ติดตามผลและควบคุมการทำงาน (Trading Dashboard / Bots Management)**
6. **แฮกตลาดด้วยปุ่มเดียว (Terminal Execution)**

---

### ขั้นตอนที่ 1: การเข้าสู่ระบบ (Login)
ใช้บัญชีผู้ใช้งานของคุณเพื่อเข้าสู่ระบบ:
- สามารถกรอก **ชื่อผู้ใช้ (Username)** หรือ **อีเมล (Email)** ของคุณได้ทั้งคู่!
- กรอกรหัสผ่านของคุณแล้วกด **"เข้าสู่ระบบ (Login)"**
*(ตัวอย่าง: หากจำอะไรไม่ได้ ให้ขอตั้งค่ารหัสผ่านเบื้องต้นจาก Admin (เช่น `123456`) เป็นต้น)*

### ขั้นตอนที่ 2: ผูกพอร์ตเตรียมเทรด (Accounts)
ก่อนที่บอทจะรู้ว่าต้องเอาออเดอร์ไปโยนทิ้งที่ไหน คุณจำเป็นต้องนำเลขบัญชีเทรด MT5 ของคุณมาฝากไว้กับเรา ผ่านระบบ MetaAPI:
1. ไปที่เมนู **"จัดการบัญชี (Accounts)"** บนซ้ายมือ
2. กดปุ่ม `+ เพิ่มบัญชี` (Add Account) มุมขวาบน
3. เลือกระบุ "ชื่อโบรกเกอร์ (Broker)", "เลขบัญชี (MT5 Number)"
4. *สำคัญที่สุด! สำเนาหมายเลข* **MetaAPI ID** มาวางแล้วกดปุ่ม "บันทึกข้อมูล"
ระบบจะซิงก์เงินทุนของคุณบนแพลตฟอร์มกับพอร์ตจริงเรียลไทม์!

---

### ขั้นตอนที่ 3: เติมเงินและอัปเกรด (Wallet & Billing)
คุณจะต้องมี **ยอดเงินจำลอง/เครดิต USD (Wallet)** เพื่อใช้จ่ายซื้อบริการทุกอย่างในนี้!

1. หากยอด Wallet (กระเป๋าขวาบน) มีน้อยเกินไป หรือเป็น 0 คุณยังเช่าบอทไม่ได้
2. ไปที่เมนู **"กระเป๋าเงิน (Wallet)"** เพื่อดูประวัติการเงิน 
3. ไปที่เมนู **"Subscriptions (อัปเกรดแพ็กเกจ)"** หากคุณต้องการสิทธิพิเศษระดับโปร! 
   - เลือกระหว่างแผน *Pro Plan* หรือ *Enterprise Plan*
   - กด **"Upgrade Now"** ระบบจะตรวจเช็กยอดเงินใน Wallet และตัดเงินให้อัตโนมัติ (สมมติระบบดึงค่าธรรมเนียม 99 USD)
   - แถบมุมซ้ายบนของคุณจะเปลี่ยนตำแหน่งสิทธิ์ใหม่ขึ้นเป็น *"ระดับ Team Lead (Enterprise)"* ทันที!

### ขั้นตอนที่ 4: ช็อปปิ้งกลยุทธ์ (Strategy Store)
คุณไม่ต้องนั่งสร้างบอทเอง แค่ไปที่คลังมหาสมบัติของเรา:

1. คลิกที่เมนู **"Strategy Store"** ด้านซ้าย
2. คุณจะเจอบริการบอทที่เก่งระดับโลก เช่น บอท Grid ยอดฮิต, ขุดรหัสทองคำ XAU, เทรดสวนกระแส Reversal
3. เมื่อเจอบอทที่ถูกใจ ให้กดปุ่ม **"ผูกเข้ากับพอร์ตตัวเอง ($49 / เดือน)"**
4. ระบบจะมีให้ **เลือกบัญชีที่คุณผูกไว้ในข้อที่ 2** นำบอทตัวนั้นวิ่งรันเข้าไปผูกติดทันที ไม่ต้องเซ็ตหลายรอบ

### ขั้นตอนที่ 5: สั่งเปิดเครื่องยนต์รัน (Trading Bots & Dashboard)
บอทที่พึ่งเช่ามา จะเข้าไปจำศิลรออยู่ในโกดังของคุณ

1. ไปเมนู **"Trading Bots"** กดปุ่ม **🔴 Play / Resume** เพื่อเปลี่ยนสถานะเป็นแถบเขียว 🟢 **"กำลังทำงาน (Running)"**
2. บอทจะเชื่อมกับ API Binance แบบ "ของจริง (Real Execution Engine)" เพื่อยิง Market/Limit ออเดอร์
3. ทุกครั้งที่คุณได้เงิน... ไปที่เมนู **"Dashboard (หน้าภาพรวม)"** เพื่อดูบัญชีทั้งหมดของคุณบวกกระจาย ทั้งกราฟ PnL โชว์ Equity รวมและลิสต์ออเดอร์วันนี้ทุกออเดอร์ในหน้าเดียว

### ขั้นตอนที่ 6 (เสริมพิเศษ): สั่งรันออเดอร์ม้ามืดด้วย Terminal
สำหรับผู้ที่อัปเกรดเป็นเวอร์ชัน Pro แล้วอยากลงมือกดออเดอร์สดให้พอร์ต!

1. ไปที่เมนู **"Terminal (เทอร์มินัลคำสั่ง)"**
2. เลือกคู่สินทรัพย์ (เช่น BTCUSDT, EURUSD) 
3. ตั้ง Lot Size และเลือกระหว่าง Market, Limit Order แล้วเหยียบคันเร่งกด  **"Buy (Long)"** หรือ **"Sell (Short)"**
4. หากคุณเป็นบัญชีระดับสูง คุณสามารถคลิกเลือกบัญชี **"All Accounts (บัญชีทั้งหมด)"** เพื่อยิงทีเดียว 3 พอร์ตรวดลอกลายแผนการเทรดขั้นเทพ!
