## ตารางสิทธิ์ตาม Role

| Role | isAdmin | isTeamLead | isAgent |
|------|---------|------------|---------|
| `user` | ❌ | ❌ | ❌ |
| `team_lead` | ❌ | ✅ | ❌ |
| `agent` | ❌ | ❌ | ✅ |
| `admin` | ✅ | ✅ | ✅ |
| `super_admin` | ✅ | ✅ | ✅ |

> **หมายเหตุ:** `isTeamLead = team_lead || isAdmin` / `isAgent = agent || isAdmin`

## 🔑 สรุปสั้น ๆ

| Role | เมนูพิเศษที่เข้าถึงได้ |
|------|-----------------------|
| `user` | เฉพาะ Trading, Reports, Wallet, Billing, Accounts, Store, Settings |
| `team_lead` | + Brokers, Groups |
| `agent` | + Agent Dashboard |
| `admin` | ทุกเมนู (รวม Admin Panel) |
| `super_admin` | ทุกเมนู เหมือน admin |
