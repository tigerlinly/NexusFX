# คู่มือการตั้งค่าระบบ Auto-Trade ฉบับรวบรัด (NexusFX)

ไฟล์นี้สรุปขั้นตอนการทำงานทั้งหมด ตั้งแต่เริ่มต้นการตั้งค่าจนถึงการเปิดบอทรันอัตโนมัติเพื่อทำกำไร

---

## 1. ตั้งค่าระบบเบื้องต้น
- **ล็อกอินเข้าใช้งาน**: เข้าสู่ระบบ NexusFX ให้เรียบร้อย
- **ผูกพอร์ตเตรียมเทรด**: ไปที่เมนู **Accounts** > **เพิ่มบัญชี (Add Account)** > เลือกระบุ โบรคเกอร์, หมายเลขพอร์ต และที่สำคัญสุดคือใส่เลข `MetaAPI ID` จากนั้นกดบันทึกข้อมูล

## 2. เติมเงินและอัปเกรด (Wallet & Subscriptions)
เพื่อให้บัญชีมีสิทธิ์เริ่มรันบอท:
- ไปที่เมนู **Wallet** เพื่อเติมเงิน/ดูยอดเงิน (USD) ซึ่งต้องใช้เป็นทุนสำหรับค่าธรรมเนียม 
- ไปที่เมนู **Subscriptions** เลือกสมัครหรืออัปเกรดแพ็กเกจ (เช่น Pro หรือ Enterprise) 

## 3. สร้างและเลือกใช้งานกลยุทธ์บอท (Bots Setup)
คุณสามารถทำให้บอทส่งคำสั่งเทรดอัตโนมัติได้ 2 วิธีหลัก:

### วิธีที่ 3.1: เลือกบอทสำเร็จรูปจากระบบ (ง่ายที่สุด)
1. ไปที่เมนู **Strategy Store** ด้านซ้าย
2. ค้นหากลยุทธ์บอทที่คุณชื่นชอบ (เช่น Grid, สวนเทรนด์)
3. กดปุ่มผูกเข้ากับพอร์ตของคุณ ระบบจะให้เลือก "บัญชีเทรด" ที่คุณเตรียมไว้ในข้อ 1 พอเลือกเสร็จบอทจะถูกเพิ่มลงคลังทันที

### วิธีที่ 3.2: รอรับออเดอร์จาก TradingView (สร้างเงื่อนไขเอง)
1. ไปที่เมนู **🤖 Bots** แล้วกด **"สร้างบอทใหม่"**
2. ตั้งชื่อบอท, เลือกบัญชีเทรด, และกำหนดรหัสความปลอดภัย `webhook_secret` 
3. คัดลอก `Webhook URL` ไปวางในหน้าต่างตั้งค่า **Alerts** ของ Indicators/Strategies ในเว็บ TradingView
4. กำหนด JSON Message เพื่อส่งค่า Buy/Sell เมื่อกราฟเข้าเงื่อนไข กลับมาที่บอทของ NexusFX

## 4. เปิดสวิตช์เริ่มต้นการทำงาน 🟢 (Start Auto Trade)
เมื่อได้บอทเข้ามาในโกดังแล้ว ขั้นตอนนี้คือการปล่อยบอทออกไปทำงาน:
1. ไปที่เมนู **Trading Bots** 
2. หารายการบอทที่เพิ่งได้รับมา (สถานะเริ่มต้นจะหยุดทำงานอยู่)
3. กดปุ่ม **▶️ Play / Resume** เพื่อสลับสถานะเป็นแถบสีเขียว **🟢 "กำลังทำงาน (Running)"**
4. ทันทีที่ขึ้นแถบเขียว บอทจะลงสนามจริงและรอจังหวะเปิด/ปิดออเดอร์ให้อัตโนมัติ!

## 5. การติดตามผล (Monitoring)
- ติดตามความเคลื่อนไหวทั้งหมดได้ที่เมนู **Dashboard** 
- คุณสามารถตรวจสอบออเดอร์ล่าสุดที่บอทเปิด รวมถึงดูกราฟ PnL โชว์ Equity ภาพรวมของพอร์ตแบบเรียลไทม์

## 6. การตั้งค่าแจ้งเตือนผ่าน Telegram (Telegram Notifications)
หากต้องการให้บอทแจ้งเตือนทุกครั้งที่มีการเปิด/ปิดออเดอร์ ให้ตั้งค่า Telegram Bot ดังนี้:

### ขั้นตอนที่ 1: การสร้าง Bot และขอ Bot Token
1. เปิดแอป Telegram แล้วค้นหา **@BotFather** (มีเครื่องหมายติ๊กถูกสีฟ้า)
2. กด `Start` หรือพิมพ์คำสั่ง `/start`
3. พิมพ์คำสั่ง `/newbot` เพื่อสร้างบอทใหม่
4. ตั้งชื่อบอท (เช่น "My NexusFX Alerts") และกดส่ง
5. ตั้ง Username ให้บอท (**ต้องลงท้ายด้วยคำว่า bot** เช่น `NexusFxAlert_bot`)
6. เมื่อสำเร็จ @BotFather จะส่งข้อความแสดงความยินดีพร้อม **HTTP API Token** (หน้าตาประมาณ `123456789:ABCDEF...`) ให้ทำการ **คัดลอก Token นี้ไว้** นำไปใส่ในช่อง `BOT TOKEN` ในระบบ

### ขั้นตอนที่ 2: การหา Chat ID (สำหรับรับข้อความส่วนตัว)
1. ในแอป Telegram ให้ค้นหา **@userinfobot** หรือ **@getmyid_bot**
2. กด `Start` หรือพิมพ์ส่งคำสั่ง `/start`
3. บอทจะตอบกลับด้วยข้อมูลบัญชีของคุณ ให้ดูตัวเลขที่ระบุว่า **Id** (เช่น `123456789`)
4. ให้นำตัวเลขนี้ไปใส่ในช่อง `CHAT ID` ในหน้าระบบ
5. **สำคัญมาก:** ก่อนทำการทดสอบส่งข้อความ คุณต้องเข้าไปที่แชทบอทที่คุณสร้างขึ้นมาเองในขั้นตอนที่ 1 แล้วกดปุ่ม **Start** (หรือส่งข้อความอะไรก็ได้ไปหาบอท 1 ครั้ง) เพื่อเป็นการเปิดสิทธิ์ให้บอทสามารถทักหาคุณได้สำเร็จ

*(หมายเหตุ: หากต้องการให้แจ้งเตือนเข้ากลุ่ม ให้ดึง Bot ที่สร้างเข้ากลุ่ม แล้วค้นหา Chat ID ของกลุ่ม ซึ่งมักจะนำหน้าด้วยเครื่องหมายลบ `-`)*
